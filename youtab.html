Youtabt

Website designed and developed as a Vanarts student project for term 3 PHP project.A high-end woman clothing brand website, complete E-Commerce activity from register/login to save order in cart and checkout. Also, on the product detail page, they able to change the colour and size of products. In the admin dashboard, manage inventory, size, colour and order status. The user enters payment information and then submits the order. Admin able to see orders, and they can send a request to delivered or cancel.
