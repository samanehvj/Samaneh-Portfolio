Youtabt

Branding /Web Development /Web design

Youtube is a high-end woman clothing brand website I completed for my PHP class while studying at Vanarts. I programmed the whole E-Commerce customer experience including customer register/login, save order to cart and checkout process. On the product detail pages, the customers are able to change the colour and size of products. Through the admin dashboard the store can manage inventory, size, colour and order status. The customer can enter their payment information and submits their orders. The store can see orders coming in, and request to deliver or cancel. The programming languages I did on this website is laravel and PHP. Some challenges, that I had to face on this project were SEO and Database. For the SEO part, it was hard to find a strong header. On the other hand, for the Database side it was very difficult to add colour and size on the product detail page.
